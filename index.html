🔧 Firebase einrichten

Für globale Zähler (Besucher, Biere aller Leute) brauchst du ein kostenloses Firebase-Konto. Ohne Firebase funktioniert der Tracker nur lokal auf dem jeweiligen Gerät.

1

Gehe zu console.firebase.google.com → Neues Projekt erstellen

2

Im Projekt: Build → Realtime Database → „Testmodus starten" wählen

3

Projekteinstellungen (⚙️) → „Deine Apps" → Web-App (</>) hinzufügen

4

Kopiere das firebaseConfig-Objekt und füge es in der index.html ein (Abschnitt „KONFIGURATION")

5

Fertig! Pushe die Datei auf GitHub Pages. 🎉
